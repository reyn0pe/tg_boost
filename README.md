## Usage
### Install
```sh
git clone https://github.com/batiscuff/tg_booster
```
### Compile and run
```sh
go build .
./tg_booster -h
```
For example: `./tg_booster -l=https://t.me/python_webscraping/30 -p=proxies.txt -w=700`
Recommended workers count(-w) - 500. Maximum workers count - 1100
